## 운영 노트 (데스크탑/터미널)
- 데스크탑 Code 탭에서 `No CLI session ID` 등 초기화 오류가 나면, **터미널 CLI가 가장 확실**:
  ```bash
  cd "D:\claud project\KinderVerse"   # 경로에 공백 있으니 따옴표 필수
  claude
  ```
- 폴더를 옮기거나 이름 바꾸면 세션이 깨진다 — 작업 중엔 경로 고정. (장기적으로 공백 없는 경로 권장: `KinderVerse`만 따로 두기.)
- 각 PR은 작게. 끝나면 변경 요약 + 스크린샷을 남기고 다음 PR로.

```
PR2. 좌측 LNB와 앱 셸을 구현해줘. SKILL.md §5 IA대로:
상단 홈·갤러리·My Board / 중단 우리반·캘린더·폴더 / 하단 프로필. 전역 진입으로 AI 채팅.
```

## PROMPT 3 (PR3) — 프롬프트바 공용화 (기존 컴포넌트 재사용)
```
PR3. 기존 로컬 프롬프트바 컴포넌트를 가져와 공용 셸 컴포넌트로 승격하고 전 페이지 하단에 상주시켜줘. 동작은 SKILL.md §7 그대로:
- 좌측 메시지 아이콘 → AI 채팅 페이지로 이동(닫힌 상태에선 펼침 트리거).
- 입력 비면 별(즐겨찾기), 있으면 전송으로 토글.
- 별 클릭 → 즐겨찾기 카드(놀이계획·놀이기록·관찰기록·문장생성·스튜디오) 솟아오름 → 클릭 시 해당 페이지 이동.
- 우측 둥근 토글 → 바 접기/펼치기.
새로 디자인하지 말고 기존 동작/스타일을 보존해 재사용해.
```
